# Examples

Real-world bot examples to help you get started with Newt.

## Available Examples

### [Hello World](./hello-world.md)
A simple bot that responds to commands. Perfect for understanding the basics.

### [Welcome Bot](./welcome-bot.md)
Automatically welcomes new members to your server with a personalized message.

### [Points Bot](./points-bot.md)
A gamification system that tracks user points and displays leaderboards.

### [Moderation Bot](./moderation-bot.md)
Content filtering and member management commands for server moderation.

## How to Use These Examples

1. Read through the example to understand what it does
2. Copy the code to a `.newt` file
3. Run it with: `newt run your-bot.newt`
4. Customize it for your needs

## Learning Path

We recommend starting with:
1. **Hello World** - Learn the basics
2. **Welcome Bot** - Understand event handlers
3. **Points Bot** - Learn about data storage
4. **Moderation Bot** - See a complete production-ready example
